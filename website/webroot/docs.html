Python Quick Start

Installation

pip install aiqa-client

Setup

Set environment variables:

export AIQA_SERVER_URL="http://localhost:3000"
export AIQA_API_KEY="your-api-key"

Basic Usage

from dotenv import load_dotenv
from aiqa import get_aiqa_client, WithTracing

# Load environment variables
load_dotenv()

# Initialize client (must be called first)
get_aiqa_client()

# Just add a decorator
@WithTracing
def my_ai_function(prompt: str):
    # Your AI code here
    return ai_model.generate(prompt)

# That's it! Now you're tracing.

JavaScript/TypeScript Quick Start

Installation

npm install @aiqa/client

Setup

Set environment variables:

export AIQA_SERVER_URL="http://localhost:3000"
export AIQA_API_KEY="your-api-key"

Basic Usage

import { getAiqaClient, withTracing } from '@aiqa/client';

// Initialize client
getAiqaClient();

// Use the decorator
@withTracing()
async function myAiFunction(prompt: string) {
  // Your AI code here
  return await aiModel.generate(prompt);
}

Go Quick Start

import "github.com/winterwell/aiqa-client-go"

Key Concepts

Tracing

AIQA uses OpenTelemetry tracing to capture function calls, inputs, outputs, and performance metrics. Each traced function creates a span that includes all relevant information.

Experiments

Run systematic experiments to test your AI across different scenarios. Compare results, track improvements, and identify regressions.

Conversations

Group related traces together using conversation IDs. This is especially useful for multi-step AI interactions or user sessions.
