Get Started

Python Quick Start

Installation

pip install aiqa-client

Setup

Set environment variables:

export AIQA_SERVER_URL="http://localhost:3000"
export AIQA_API_KEY="your-api-key"

Basic Usage

from dotenv import load_dotenv
from aiqa import get_aiqa_client, WithTracing

# Load environment variables
load_dotenv()

# Initialize client (must be called first)
get_aiqa_client()

# Just add a decorator
@WithTracing
def my_ai_function(prompt: str):
    # Your AI code here
    return ai_model.generate(prompt)

# That's it! Now you're tracing.

JavaScript/TypeScript Quick Start

Installation

npm install @aiqa/client

Setup

Set environment variables:

export AIQA_SERVER_URL="http://localhost:3000"
export AIQA_API_KEY="your-api-key"

Basic Usage

import { getAiqaClient, withTracing } from '@aiqa/client';

// Initialize client
getAiqaClient();

// Use the decorator
@withTracing()
async function myAiFunction(prompt: string) {
  // Your AI code here
  return await aiModel.generate(prompt);
}

Tracing

AIQA uses OpenTelemetry tracing to capture function calls, inputs, outputs, and performance metrics. Each traced function creates a span that includes all relevant information.

Conversations

Group related traces together using conversation IDs. This is especially useful for multi-step AI interactions or user sessions.

Run experiments

After traces are flowing, use the AIQA app to run systematic experiments across scenarios: compare prompts, models, or inputs, track improvements over time, and catch regressions. You can drive experiments from code with the same Python and JavaScript clients: create a dataset in the app, define metrics, then use ExperimentRunner to execute your model on each example and upload scores.

Python (aiqa-client)

import asyncio
from aiqa import ExperimentRunner

async def main():
    runner = ExperimentRunner(
        dataset_id="YOUR_DATASET_ID",
        organisation_id="YOUR_ORGANISATION_ID",
    )
    runner.create_experiment({"name": "My experiment"})

    # The runner needs a function to call your model. It will pass in the Example input data and any parameters.
    async def my_engine(input_data, parameters):
        return await my_model(input_data, parameters)

    await runner.run(my_engine)
	
    print(runner.get_summaries())

asyncio.run(main())

JavaScript / TypeScript (@aiqa/client-js)

Install npm install @aiqa/client-js, set the same environment variables, then use ExperimentRunner from the package. Pass datasetId and organisationId, call createExperiment, then await runner.run(myEngine, scorer). The optional scorer(output, example) returns numeric metric scores. Use getSummaryResults() for aggregate stats. For more control, call runExample in a loop over getExampleInputs().

import { ExperimentRunner } from '@aiqa/client-js';

async function main() {
  const runner = new ExperimentRunner({
    datasetId: 'YOUR_DATASET_ID',
    organisationId: 'YOUR_ORGANISATION_ID',
  });
  await runner.createExperiment({ name: 'My experiment' });

  async function myEngine(input) {
    return await runYourModel(input);
  }
  async function scorer(output, example) {
    return { /* metricIdOrName: score */ };
  }

  await runner.run(myEngine, scorer);
  console.log(await runner.getSummaryResults());
}

main();

MCP server

Prefer to stay in your editor? We host an MCP server so assistants such as Cursor or Claude Code can list traces, datasets, and experiments using the same API you use in the app—handy when you are debugging or designing evals next to your code.

  1. In the AIQA app, open API Keys and create a key with the permissions you need.
  2. In your client’s MCP settings, add an HTTP/SSE server pointed at our endpoint and pass your key in the Authorization header (Bearer token).

Example for Cursor (mcpServers):

{
  "mcpServers": {
    "aiqa": {
      "url": "https://mcp-aiqa.winterwell.com/sse",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}

Replace YOUR_API_KEY with your real key. Any MCP client that supports HTTP/SSE can use the same URL and headers.
